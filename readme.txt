== Description == 

WS Custom Scrollbar plugin will enable change scrollbar styles where you can change scrollbar color, border radius, scroll speed, width, hide delay & other settings.

**Features** 

* Simple installation and activation
* Show/Hide Styled Scrollbar
* Change Scrollbar Width
* Change Scrollbar Scroll Speed
* Change Scrollbar Radius
* Change Scrollbar Background Color
* Change Scrollbar Border Color
* Change Scrollbar Opacity
* Enable/Disable Scrollbar Autohide
* All Browsers Support!

== Installation ==  

1. Upload 'ws-custom-scrollbar' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Change Option to Setting->WS Custom Scrollbar Page
4. Change options and injoy!
